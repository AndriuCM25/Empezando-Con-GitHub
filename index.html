📌 Introducción

GitHub es una de las plataformas más importantes para desarrolladores. Permite guardar proyectos, trabajar en equipo y controlar versiones del código.

En esta guía aprenderás desde cero cómo empezar a usarlo de forma sencilla.

💡 ¿Qué es GitHub?

GitHub es una plataforma basada en Git que permite almacenar proyectos de programación en la nube y colaborar con otras personas.

🚀 Pasos para empezar

1
Crear una cuenta en GitHub desde su página oficial.
2
Instalar Git en tu computadora.
3
Crear tu primer repositorio.
4
Subir archivos usando comandos Git.
Ir a GitHub

⌨️ Comandos básicos de Git

Inicializar repositorio:

git init

Agregar archivos:

git add .

Guardar cambios:

git commit -m "Primer commit"

Subir a GitHub:

git push origin main

⭐ Beneficios de GitHub

Trabajo en equipo

Permite que varios desarrolladores trabajen en un mismo proyecto.

Control de versiones

Puedes ver cambios y regresar a versiones anteriores.

Seguridad

Tu código se guarda en la nube de forma segura.

Portafolio

Puedes mostrar tus proyectos a empresas o profesores.
