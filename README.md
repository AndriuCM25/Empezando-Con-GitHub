# Empezando-Con-GitHub

# Iniciando en GitHub

Este repositorio contiene una guía básica para aprender a usar Git y GitHub desde cero. Está dirigido a principiantes que desean entender el control de versiones y la gestión de proyectos en GitHub.

---

## ¿Qué es GitHub?

GitHub es una plataforma en la nube que permite:

- Guardar proyectos de programación
- Trabajar en equipo
- Controlar versiones del código
- Compartir proyectos con otros desarrolladores

GitHub utiliza Git como sistema de control de versiones.

---

## ¿Qué es Git?

Git es una herramienta de control de versiones que permite:

- Registrar cambios en el código
- Volver a versiones anteriores
- Trabajar con ramas (branches)
- Fusionar cambios de diferentes desarrolladores
